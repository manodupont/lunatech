# Lunactech E-Shop Web App #

### Prerequisites ###
* Node 8.X

### How to ###

# Local dev 
## Build
1. `npm install`
2. `npm run build:dev`

## Test
1. `npm install`
2. `npm test`

## Start
1. `npm run build:dev`
2. `npm run start:client`
3. `npm run start:server`
4. http://localhost:8080

# Build with Docker
## Build and Run
1. `docker build -t <image_name>`
2. `npm run start`
